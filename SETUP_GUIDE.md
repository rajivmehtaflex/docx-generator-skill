## 🎯 How to Use

### 1. Web Interface (Easiest)
1. Open `index.html` in your browser
2. Type or paste Markdown/HTML
3. Configure options (title, author, etc.)
4. Click "Generate DocX"
5. Download your document

### 2. Python Library (For Codex)
```python
from docx_generator import generate_docx_from_markdown

markdown_content = """
# My Report

**Key Findings:**
- Revenue increased 25%
- Customer satisfaction up 30%

| Metric | Q4 | Q1 |
|--------|-----|-----|
| Sales | $1M | $1.25M |
| NPS   | 45  | 58  |
"""

options = {
    "title": "Q4 Business Report",
    "author": "AI Analyst",
    "header": "Confidential",
    "footer": "Generated by DocX Generator",
    "page_number_format": "total"
}

docx_base64 = generate_docx_from_markdown(markdown_content, options)
# Use base64 content for download or storage
```

### 3. Advanced Customization
```python
from docx_generator import DocxGenerator

# Create professional document
generator = DocxGenerator()
generator.create_document(
    title="Annual Report 2024",
    author="Finance Department"
)

# Add cover page
generator.add_cover_page(
    title="Annual Report 2024",
    subtitle="Financial Performance Summary",
    author="Finance Department",
    date="January 2024"
)

# Add custom headers/footers
generator.add_header("Confidential Document", alignment="center")
generator.add_footer("Internal Use Only", alignment="center")
generator.add_page_numbers(format_type="total")

# Add content
html_content = """
<h1>Executive Summary</h1>
<p>Our company achieved remarkable growth in 2024...</p>
<h2>Key Metrics</h2>
<table>
    <tr><th>Metric</th><th>2023</th><th>2024</th><th>Change</th></tr>
    <tr><td>Revenue</td><td>$10M</td><td>$12.5M</td><td>+25%</td></tr>
    <tr><td>Profit</td><td>$2M</td><td>$3M</td><td>+50%</td></tr>
</table>
"""
generator.process_html_content(html_content)

# Add table of contents
generator.add_table_of_contents()

# Get result
docx_base64 = generator.save_to_base64()
```

## 🔧 Configuration Options

### Basic Options
```python
options = {
    "title": "Document Title",
    "author": "Author Name",
    "header": "Header Text",
```

## 🐛 Troubleshooting

### "Loading Python packages..." stuck
**Solution**: Use local server, not file:// protocol
```bash
python -m http.server 8000
```

### Memory errors on large documents
**Solution**: Split into smaller documents (<50 pages each)

### Formatting not applied correctly
**Solution**: Use simpler HTML/Markdown, test incrementally

### Download fails
**Solution**: Check browser popup settings, try different browser

## 📊 Performance Tips

1. **Initial Load**: Wait 5-8 seconds for WASM initialization
2. **Document Size**: Keep under 50 pages for best performance
3. **Complex Tables**: Simplify for faster generation
4. **Images**: Limit size and number
